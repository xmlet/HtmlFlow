This release introduces `TBody` and `Tr` as valid `HtmlPage` root elements and adds root builder extensions for `tbody` and `tr`, along with comprehensive tests.

{/_ truncate _/}

This feature enables the creation of HTML fragments that can be embedded in other HTML documents, such as tables, without requiring a complete HTML structure.
