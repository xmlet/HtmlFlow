This release includes the following fixes and updates:

- Resolved [Issue 103](https://github.com/xmlet/HtmlFlow/issues/103). Using `.text()` interleaved with attributes is no longer supported and will now throw an exception. {/_ truncate _/}
- Resolved [Issue 104](https://github.com/xmlet/HtmlFlow/issues/104). Disabling indentation incorrectly removed newlines inside text and script blocks. This has been corrected.
- Enhanced the documentation with additional examples of data binding, conditionals (if/else), and loops.
- `HtmlView` is now strongly typed with the type `M` corresponding to the model.
